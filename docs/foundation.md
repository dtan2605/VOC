# Project Foundation (Phase 0)

## Mục tiêu

Thiết lập nền tảng vững chắc cho hệ thống học tiếng Anh, đảm bảo các thành phần chính sẵn sàng để phát triển tiếp theo.

## Kiến trúc cơ bản

* Frontend: React + TypeScript
* Backend: ASP.NET Core microservices
* AI Services: FastAPI
* Messaging: RabbitMQ
* Cache: Redis
* Database: MySQL
* Object storage: MinIO
* Containerization: Docker
* DevOps: Docker Compose, GitHub Actions
* Documentation: README, Architecture, SRS, API contract
